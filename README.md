# Splashscreen-Holztechnik
Splash Screen


Der Splashscreen „Holztechnik Lernwelt“ fungiert als zentraler Start- und Eingangsbildschirm für das digitale Lernspiel HOLZEN.

In seiner aktuellen Form dient der Splashscreen als Einstieg in das Spiel HOLZEN, indem er den Übergang vom Aufruf der Webanwendung zur eigentlichen Spielumgebung gestaltet. Gleichzeitig ist er so konzipiert, dass er nicht nur auf ein einzelnes Spiel beschränkt bleiben muss.

Langfristig ist vorgesehen, die „Holztechnik Lernwelt“ zu einem übergeordneten Portal auszubauen. Über diesen Startbildschirm sollen perspektivisch weitere Spiele, thematische Erweiterungen oder unterschiedliche Lernlevel integriert und aufgerufen werden können. Der Splashscreen übernimmt damit die Funktion einer zentralen Navigations- und Auswahloberfläche für verschiedene digitale Lernangebote im Bereich Holztechnik.

Durch diese modulare Struktur wird es möglich, bestehende Inhalte zu erweitern, neue Lernformate zu ergänzen und das System schrittweise auszubauen, ohne die grundlegende Architektur verändern zu müssen.
